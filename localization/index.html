Localization

2017-03-13

There are several ways to help with localization of Violentmonkey.

Pull Requests

This way always works and is recommended at the moment.

  1. Fork Violentmonkey from GitHub.
  2. Copy src/_locales/en/messages.yml into src/_locales/<your_locale>/messages.yml.
  3. Translate messages in the created messages.yml or the one youâ€™d like to modify.
    Make sure the modified messages.yml is valid yaml.
    If you have Node.js 8+ installed, you can check and reformat it by running npm run i18n.
  4. Commit changes and create a pull request.
